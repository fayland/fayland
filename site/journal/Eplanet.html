Eplanet

Category: Script   Keywords: Eplanet

我写 Eplanet 的目的是用于在本地/localhost上写文章并且创建本地html文档，然后通过ftp上传来达到网页更新目的。
此法多用于使用免费主机而没有cgi-bin权限空间的朋友。
本程序也适用与在服务器端使用，但未过滤任何input，需要的朋友可以以此为蓝本进行二次开发。
Eplanet由Perl+Mysql（其他的数据库当然也可以）组成。详细的说明见压缩包里的 README
