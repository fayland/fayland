RSS date

Category: Diary   Keywords: RSS

上个礼拜车东兄跟我说起我的 RSS 似乎时间不对。也一直没什么时间去改程序（这个学期一星期课有三十几节，每晚都要去上晚自习复习考研，还真是没什么时间）。今天去车站买了十一去郑州的火车票，然后就抽个空看看了 XML::RSS

发现原来是在 item 里没有加 dc:date. 修改下倒也及其容易： 

$rss->add_item(
    # ...
    dc => {
        date     => $_create_date,
    },
);
似乎这样就可以了。其他也先不管了。

最近虽然考研也挺忙的，却也一直学习 Perl. 毕竟以后可能要靠这个吃饭。就是自己写代码写得太少，总感觉手生。看着 chunzi 兄用 Catalyst 开发了个 PerlChina Member, 觉得挺不错的。有时候倒也挺想一起合作开发下，想积累点经验。毕竟光看和写代码纯粹是两码子事。
