在 Win32 下的 SourceForge Shell/CVS 登陆

Category: Miscellaneous   Keywords: sourceforge ssh cvs

最近因为参加了 Alexa 的 ShellWeb, 所以要跟 http://www.sourceforge.net 打交道。以前没在 SourceForge 参加过任何项目，所以这是第一次碰到这么复杂的东西。在 Baidu 上搜索了一下。

Updated

TortoiseCVS 非常不错。用这个真是简单的没话说。内置了 PuTTY, 本身就在 sf.net 上开发的，对 sf.net 支持很好。
如果在 Win32 下，在 sf.net 上有项目要协作的话，这个是很好的选择。

Abondoned

下面是某些连接：
  • 英文的 SourceForge.net: CVS (Version Control for Source Code) http://sourceforge.net/docs/E04/
  • 英文的 SourceForge.net: SSH Key Generation and Usage http://sourceforge.net/docs/F02/
  • 英文的 Config Wincvs For SourceForge http://dev.csdn.net/article/35/article/64/64175.shtm
  • PuTTY downloads http://www.chiark.greenend.org.uk/~sgtatham/putty/download.html
按照那些说明用 puttygen.exe 弄了 SSH-2 DSA key (包括shell 和 cvs 的两个）
在 sf.net/account 里 update 了 [Edit SSH Keys for Shell/CVS]
也不知道成功了没，sf.net 是批量更新 ssh key 的，所以我得过几个小时才知道成功了没有。
到时候再写或者再试验。
