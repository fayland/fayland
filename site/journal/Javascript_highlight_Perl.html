Javascript Code Syntax Highlighter / Perl

Category: Javascript   Keywords: SyntaxHighlighter

最初相识 dp.SyntaxHighlighter 在 chunzi 兄的 blog 上（dp.SyntaxHighlighter）。
觉得挺棒的。最重要是不占用服务器资源。（以前倒是写过一篇 用 Syntax::Highlight::Perl 将Perl代码着色）
去 http://www.dreamprojections.com/syntaxhighlighter/ 下载过来看了看源码，就依样划葫芦写了个 shBrushPerl.js
不过麻烦的是 Javascript 的 RegExp 似乎挺难相处的。\ 这样似乎无法转义。我也不知具体该如何操作。

一个简单的演示位于 http://www.fayland.org/journal/SyntaxHighlight/perl.html
若想下载的话直接将它另存为或者去 http://www.fayland.org/journal/SyntaxHighlight/ 拖走。

若有所改进，请发 email 与我。谢谢。
